| ------------------------- | ------------------------ | -------------------- | ---------------------------------------------------------- | --------- |
| **1. 猜发音能力训练**     | 提升“看到单词会读”的能力 | 不再一看到单词就懵   | ① 遇到生词先**自己猜发音** ② 用词典验证 ③ 听原声 ④ 跟读5遍 | 15 min    |

# 详细执行方法

## ① 猜发音训练（最重要）

目标：

> 建立 **看到单词 → 猜读音 → 验证** 的能力。

流程：

| Step   | 操作                     |
| ------ | ------------------------ |
| Step 1 | 遇到生词先**自己猜读音** |
| Step 2 | 用欧路词典/剑桥词典查    |
| Step 3 | 听美音/英音              |
| Step 4 | 模仿跟读 5 次            |

例子：

| 单词        | 先猜 | 正确           |
| ----------- | ---- | -------------- |
| enough      | ?    | /ɪˈnʌf/        |
| apartment   | ?    | /əˈpɑːrtmənt/  |
| comfortable | ?    | /ˈkʌmfərtəbəl/ |

------

## ② 音标重点训练（不要全背）

只练最影响口语的几个。

| 音标        | 常见问题         | 示例                |
| ----------- | ---------------- | ------------------- |
| /ɪ/ vs /iː/ | 长短音混淆       | ship / sheep        |
| /æ/ vs /e/  | 容易读成中文音   | bad / bed           |
| /ʌ/         | 亚洲学习者常错   | enough / country    |
| /ə/ (schwa) | 英语最重要弱读音 | apartment / support |

------

## ③ Shadowing（非常适合你）

目标：

> 提升 **发音 + 节奏 + 自动输出**。

素材：

**Interchange Level 2 Audio**

流程：

| Step | 做法               |
| ---- | ------------------ |
| 1    | 听一句（不看文本） |
| 2    | 看文本确认内容     |
| 3    | 跟读模仿           |
| 4    | 与音频同步说       |

重点模仿：

- 重音
- 连读
- 节奏
- 语调

------

## ④ 大声朗读训练

目标：

> 解决“读文章卡壳”。

规则：

| 做法             | 说明         |
| ---------------- | ------------ |
| 大声读           | 不要默读     |
| 遇到不会的词停下 | 查词 + 修正  |
| 再重新读句子     | 强化正确发音 |

推荐素材：

- Interchange课文
- Reading Explorer
- 简短新闻
- 分级读物

------

## ⑤ Pronunciation Notebook（发音错词本）

建立自己的易错词表。

格式：

| 单词        | 我的错误       | 正确发音       | 复习日期 |
| ----------- | -------------- | -------------- | -------- |
| comfortable | com-for-ta-ble | /ˈkʌmfərtəbəl/ | 5/20     |
| privacy     | pri-va-cy      | /ˈprɪvəsi/     | 5/20     |
| convenient  | con-ve-ni-ent  | /kənˈviːniənt/ | 5/21     |

# 你的每日训练版本（推荐）

| 时间      | 内容                        |
| --------- | --------------------------- |
| 15 min    | 猜发音 + 查词 + 跟读        |
| 15–20 min | Interchange Audio Shadowing |
| 10 min    | 大声朗读                    |
| 5 min     | 错词复习                    |

**总计：45–50分钟/天**

# 30天后的预期变化

| 训练前       | 训练后         |
| ------------ | -------------- |
| 看单词不会读 | 能猜出大概发音 |
| 阅读频繁卡壳 | 流畅度明显提高 |
| 单词经常读错 | 高频错误减少   |
| 一开口紧张   | 输出更自动化   |

这个计划和你现在的 **Interchange Level 2 + 英语课** 非常匹配。
